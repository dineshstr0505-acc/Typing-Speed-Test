A simple and responsive Typing Speed Test web application built using *HTML, CSS, and JavaScript*. 
This project helps users test and improve their typing speed and accuracy by typing a randomly displayed paragraph within a limited time.

Features:
- Random typing paragraphs
- Real-time typing speed (WPM)
- Accuracy calculation
- Mistake counter
- Countdown timer
- Restart test option
- Responsive and user-friendly interface

Technologies Used:
- HTML5
- CSS3
- JavaScript
